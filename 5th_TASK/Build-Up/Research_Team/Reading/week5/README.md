**📢 5주차 Final Session: Review & Change**


4주간 진행했던 핵심 주제들을 서로 바꿔서 발제하며, 지식을 완전히 내 것으로 만드는 최종 마무리 세션입니다. 

이번 주차는 별도의 GitHub 코드 제출 없이, 노션에 **발표 자료(정리 노트)** 를 제출하는 것으로 Task를 대신합니다.

#### 🗓️ Agenda: "역할 체인지 발제"

방식: 사다리타기로 결정된 새로운 담당 주제에 대해 5~10분 내외의 핵심 요약 발제 진행

목표: 본인이 담당하지 않았던 논문을 깊이 있게 이해하고, 팀원들에게 자신만의 언어로 다시 설명하기 !

#### 📋 발제 담당자 및 주제 배정

| 주제 | 담당자 | 메인 논문 | 발제 핵심 포인트 |
| :--- | :---: | :--- | :--- |
| **ResNet** | **태영** | [Deep Residual Learning](https://arxiv.org/abs/1512.03385) | Skip Connection을 통한 최적화 문제 해결 및 층의 깊이와 성능의 관계 |
| **NCF** | **윤지** | [Neural Collaborative Filtering](https://arxiv.org/abs/1708.05031) | MF의 선형적 한계 극복 및 MLP를 이용한 비선형 특징 추출 방식 |
| **RNN** | **서연** | [RNN & Sequence Modeling](https://arxiv.org/abs/1406.1078) | Hidden State를 통한 정보 전달 구조 및 Gradient Vanishing/Exploding 문제 |
| **GAN** | **병조** | [Generative Adversarial Nets](https://arxiv.org/pdf/1406.2661) | 생성자와 판별자의 Minimax Game 원리 및 적대적 학습 메커니즘 |

(발제 핵심 포인트는 예시로 작성한 것이므로 다른 부분을 리뷰해주셔도 좋습니다!)
