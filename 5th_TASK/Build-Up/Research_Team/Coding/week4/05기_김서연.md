# Model Architecture
## Generator
#### 역할
랜덤 노이즈를 입력받아 가짜 이미지를 생성하는 모델

#### 구조
DCGAN에서는 ConvTranspose2d(Deconvolution)을 사용하여 노이즈 벡터를 점점 이미지 형태로 확장함
```
class Generator(nn.Module):
    def __init__(self, ngpu):
        super(Generator, self).__init__()
        self.main = nn.Sequential(
            nn.ConvTranspose2d(nz, ngf * 8, 4, 1, 0, bias=False),
            nn.BatchNorm2d(ngf * 8),
            nn.ReLU(True),

            nn.ConvTranspose2d(ngf * 8, ngf * 4, 4, 2, 1, bias=False),
            nn.BatchNorm2d(ngf * 4),
            nn.ReLU(True),

            nn.ConvTranspose2d(ngf * 4, ngf * 2, 4, 2, 1, bias=False),
            nn.BatchNorm2d(ngf * 2),
            nn.ReLU(True),

            nn.ConvTranspose2d(ngf * 2, ngf, 4, 2, 1, bias=False),
            nn.BatchNorm2d(ngf),
            nn.ReLU(True),

            nn.ConvTranspose2d(ngf, nc, 4, 2, 1, bias=False),
            nn.Tanh()
        )
```
#### 핵심 포인트
- 입력: 랜덤 노이즈 z
- ConvTranspose2d -> 이미지 크기 증가
- BatchNorm -> 학습 안정화
- ReLU -> 활성화 함수
- 마지막 Tanh -> 이미지 픽셀 범위 조정

## Discriminator
#### 역할
입력 이미지가 진짜인지 가짜인지 판별

#### 구조
Conv2d를 사용하여 이미지 특징을 추출하고 마지막에 Sigmoid로 진짜 확률을 출력
```
class Discriminator(nn.Module):
    def __init__(self, ngpu):
        super(Discriminator, self).__init__()
        self.main = nn.Sequential(
            nn.Conv2d(nc, ndf, 4, 2, 1, bias=False),
            nn.LeakyReLU(0.2, inplace=True),

            nn.Conv2d(ndf, ndf * 2, 4, 2, 1, bias=False),
            nn.BatchNorm2d(ndf * 2),
            nn.LeakyReLU(0.2, inplace=True),

            nn.Conv2d(ndf * 2, ndf * 4, 4, 2, 1, bias=False),
            nn.BatchNorm2d(ndf * 4),
            nn.LeakyReLU(0.2, inplace=True),

            nn.Conv2d(ndf * 4, ndf * 8, 4, 2, 1, bias=False),
            nn.BatchNorm2d(ndf * 8),
            nn.LeakyReLU(0.2, inplace=True),

            nn.Conv2d(ndf * 8, 1, 4, 1, 0, bias=False),
            nn.Sigmoid()
        )
```
#### 핵심 포인트
- Conv2d -> 이미지 특징 추출
- LeakyReLU -> dying ReLU 문제 완화
- BatchNorm -> 학습 안정화
- 마지막 Sigmoid -> 진짜 확률 출력

# Loss & Optimizer
- GAN은 이진 분류 문제로 학습되기 때문에 Binary Cross Entropy Loss를 사용함
```
criterion = nn.BCELoss()
```
- Optimizer은 Adam을 사용
```
optimizerD = optim.Adam(netD.parameters(), lr=0.0002, betas=(0.5, 0.999))
optimizerG = optim.Adam(netG.parameters(), lr=0.0002, betas=(0.5, 0.999))
```
#### 핵심
- optimizerD -> Discriminator 업데이트
- optimizerG -> Generator 업데이트

# Training Loop
- GAN 학습의 핵심은 두 모델을 번갈아 학습하는 것

### Step 1: Discriminator 학습
#### 목표
진짜 이미지와 가짜 이미지를 구별하도록 학습
```
netD.zero_grad()

# real image
output = netD(real_images)
errD_real = criterion(output, real_label)

# fake image
fake = netG(noise)
output = netD(fake.detach())
errD_fake = criterion(output, fake_label)

errD = errD_real + errD_fake
errD.backward()
optimizerD.step()
```
#### 핵심
- 진짜 -> label = 1
- 가짜 -> label = 0
- Discriminator가 진짜 / 가짜 구별하도록 학습

### Step 2: Generator 학습
#### 목표
가짜 이미지를 진짜처럼 보이게 만들기
```
netG.zero_grad()

output = netD(fake)
errG = criterion(output, real_label)

errG.backward()
optimizerG.step()
```
#### 핵심
- 가짜 이미지인데 lable = 1로 설정
- 이유 -> Discriminator를 속이도록 학습

---

# Overall Training Flow
GAN 학습 과정은 다음 순서로 반복됨
1. 랜덤 노이즈 생성
2. Generator가 가짜 이미지 생성
3. Discriminator가 진짜/가짜 판별 학습
4. Generator가 Discriminator를 속이도록 학습
5. 반복

- Discriminator -> 구별 연습
- Generator -> 속이기 연습
