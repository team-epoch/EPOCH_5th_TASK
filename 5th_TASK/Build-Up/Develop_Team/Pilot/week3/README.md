# 1️⃣ TASK 1 — 모델링 (Modeling)

전처리된 데이터와 파생 변수를 활용하여 **머신러닝 모델을 학습**합니다.

## 진행 방법

- 분석 목적에 맞게 **문제 유형을 정의**합니다.
> Classification (분류) </br>
> Regression (회귀)


- 문제 유형에 맞는 모델을 선택합니다.

### 예시 모델

**분류 모델**

- Logistic Regression
- Decision Tree
- Random Forest
- XGBoost

**회귀 모델**

- Linear Regression
- Ridge / Lasso
- Random Forest Regressor

---

## 모델 학습 과정

다음 과정을 포함하여 모델을 학습합니다.

- **Train / Test Split** 수행
- 필요 시 **Cross Validation 적용**
- 모델 학습 과정 정리

또한 다음 내용을 간단히 기록해 주세요.

- 모델 선택 이유
- 하이퍼파라미터 설정 이유
- 학습 시간 또는 난이도

---

# 2️⃣ TASK 2 — 모델 평가 (Model Evaluation)

학습한 모델을 **성능 지표를 통해 평가하고 비교**합니다.

## 평가 지표 예시

### 분류 문제

- Accuracy
- Precision
- Recall
- F1-score
- ROC-AUC

### 회귀 문제

- MSE
- MAE
- RMSE
- R²

---

## 평가 내용

다음 내용을 포함하여 모델을 평가합니다.

- 각 평가 지표의 **의미 설명**
- 모델 간 **성능 비교**
- 모델 해석 관점 분석

가능하다면 다음도 포함해 주세요.

- **Feature Importance**
- **계수 해석**
- **오분류 분석**
- **잔차 패턴 분석**

---

## 시각화 (선택)

다음과 같은 시각화를 추가하면 좋습니다.

- Confusion Matrix
- ROC Curve
- Feature Importance Plot
- Residual Plot
