## 데이터 준비
추천 시스템에서는 user와 movie ID를 embedding layer에서 사용할 수 있는 index 형태로 변환해야 함
#### create_dataset()
```
def create_dataset(ratings, top=None):
    unique_users = ratings.userId.unique()
    user_to_index = {old: new for new, old in enumerate(unique_users)}
    new_users = ratings.userId.map(user_to_index)
    
    unique_movies = ratings.movieId.unique()
    movie_to_index = {old: new for new, old in enumerate(unique_movies)}
    new_movies = ratings.movieId.map(movie_to_index)
    
    n_users = unique_users.shape[0]
    n_movies = unique_movies.shape[0]
    
    X = pd.DataFrame({'user_id': new_users, 'movie_id': new_movies})
    y = ratings['rating'].astype(np.float32)
    
    return (n_users, n_movies), (X, y), (user_to_index, movie_to_index)
```
- userid, movieid는 embedding layer에서 사용할 수 없음
- 그래서 index로 변환
#### 예시

userId 15 → index 0

movieId 260 → index 0
movieId 296 → index 1

## 핵심 모델 구조
추천 시스템 모델은 user embedding과 item embedding을 학습하여 interaction을 예측하는 구조
```
(user_id, movie_id)
        │
        ▼
User Embedding
Movie Embedding
        │
        ▼
Concatenate
        │
        ▼
Dropout
        │
        ▼
Hidden Layer (Linear + ReLU)
        │
        ▼
Linear
        │
        ▼
Sigmoid
        │
        ▼
Predicted Rating
```
- user와 movie를 embedding vector로 변환
- 두 embedding을 결합하여 user-item interaction 학습
- 최종적으로 사용자가 해당 영화에 줄 평점 예측
---
## 모델 코드
#### EmbeddingNet
```
class EmbeddingNet(nn.Module):

    def __init__(self, n_users, n_movies,
                 n_factors=50,
                 embedding_dropout=0.02,
                 hidden=10,
                 dropouts=0.2):

        super().__init__()

        hidden = get_list(hidden)
        dropouts = get_list(dropouts)

        self.u = nn.Embedding(n_users, n_factors)
        self.m = nn.Embedding(n_movies, n_factors)

        self.drop = nn.Dropout(embedding_dropout)

        self.hidden = nn.Sequential(
            nn.Linear(n_factors*2, hidden[0]),
            nn.ReLU(),
            nn.Dropout(dropouts[0])
        )

        self.fc = nn.Linear(hidden[-1], 1)
```
#### 주요 구성 요소
- nn.Embedding → user / movie vector 생성
- Dropout → 과적합 방지
- Linear + ReLU → interaction 학습
- Linear → 최종 평점 출력

## Forward 흐름
forward 과정
```
(user_id, movie_id)
        │
        ▼
User Embedding + Movie Embedding
        │
        ▼
Concatenate
        │
        ▼
Dropout
        │
        ▼
Hidden Layer
        │
        ▼
Output Layer
        │
        ▼
Sigmoid
        │
        ▼
Predicted Rating
```
#### 코드
```
features = torch.cat([self.u(users), self.m(movies)], dim=1)

x = self.drop(features)

x = self.hidden(x)

out = torch.sigmoid(self.fc(x))
```
#### 단계
1. self.u(users) → user embedding 생성
2. self.m(movies) → movie embedding 생성
3. torch.cat() → 두 embedding 결합
4. hidden layer에서 user-item interaction 학습
5. sigmoid를 통해 최종 평점 예측

## 학습 과정
추천 모델은 평점 예측 문제이므로 regression 문제로 학습됨
#### Loss
```
criterion = nn.MSELoss()
```
MSE Loss를 사용하여 예측 평점과 실제 평점 차이 최소화
#### Optimizer
```
optimizer = optim.Adam(net.parameters(), lr=lr)
```
Adam optimizer를 사용하여 모델 파라미터 업데이트
#### Training Loop
```
for epoch in range(n_epochs):

    for phase in ('train', 'val'):

        for batch in batches(*datasets[phase]):

            x_batch, y_batch = batch

            optimizer.zero_grad()

            outputs = net(x_batch[:,0], x_batch[:,1])

            loss = criterion(outputs, y_batch)

            if phase == "train":
                loss.backward()
                optimizer.step()
```

## 평가 방식
추천 시스템에서는 두 가지 방식의 평가가 존재
#### Explicit Feedback
사용자가 직접 평가한 데이터를 이용
- 영화 평점
- 리뷰 점수
##### 대표 평가 지표
```
RMSE
```
값이 작을수록 예측 성능이 좋음
#### Implicit Feedback
사용자의 행동 데이터를 이용
- 클릭
- 구매
- 시청 기록
##### 대표 평가 지표
- Hit Rate
- AUC
- NDCG

## 논문 아이디어 -> 코드 반영
Neural Collaborative Filtering의 핵심 아이디어는 user와 item을 embedding vector로 표현하고 이를 통해 user-item interaction을 neural network로 학습하는 것
#### Embedding
```
self.u = nn.Embedding(n_users, n_factors)
self.m = nn.Embedding(n_movies, n_factors)
```
- user/item을 dense vector로 변환
#### Interaction 학습
```
features = torch.cat([self.u(users), self.m(movies)], dim=1)
```
- user embedding + item embedding 결합
#### Neural Network 기반 interaction 학습
```
self.hidden = nn.Sequential(
    nn.Linear(n_factors*2, hidden[0]),
    nn.ReLU(),
    nn.Dropout(dropouts[0])
)
```
즉, 기존 Matrix Factorization을 Neural Network 기반으로 확장한 구조라고 볼 수 있음

- dropout과 hidden layer를 활용하여 추천 모델을 deep learning 방식으로 구현한 점
