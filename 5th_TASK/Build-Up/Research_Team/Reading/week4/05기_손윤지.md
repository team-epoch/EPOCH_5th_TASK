**📖 Abstract**

> ℹ️ 본인의 방식으로 재해석 해주세요. 그대로 가져오는 것은 금합니다.
> 

이 논문은 새로운 생성모델 학습 방법인 Generative Adversarial Nets(GAN) 를 제안한다. GAN은 두 개의 신경망을 서로 경쟁시키는 방식으로 학습된다. 하나는 실제 데이터와 유사한 샘플을 생성하는 생성자(Generator) 이고, 다른 하나는 입력된 데이터가 실제 데이터인지 생성자가 만든 가짜 데이터인지를 판별하는 판별자(Discriminator)이다. 

생성자는 판별자를 속이는 방향으로 학습되고, 판별자는 이를 더 정확히 구분하는 방향으로 학습된다. 이러한 대립적 구조를 통해 생성자는 점차 실제 데이터의 분포를 잘 따라하게 된다. 저자들은 이 방법이 이론적으로도 적합하며, 실험적으로도 생성모델로서 가능성이 있음을 보였다. 이 논문은 이후 다양한 생성모델 연구의 출발점이 되었다는 점에서 의의가 있다.

---

**📚 Background**

> ℹ️ 논문의 주제와 관련된 기존 연구들 및 배경 지식을 정리
> 

**📍 Related Work 1: 확률 중점 생성모델 연구**

기존 Generative Model들에 대해선 데이터를 생성하기 위해 확률분포를 직접 모델링하는 방식이 많이 사용됐다. Boltzmann Machine와 같은 모델들은 데이터의 분포를 학습하고 이를 바탕으로 새로운 샘플을 생성하려고 하였다. 하지만 이러한 방법들은 학습 과정에서 복잡한 확률 계산이나 Markov Chain Monte Carlo(MCMC) 같은 샘플링 기법이 필요한 경우가 많아서, 학습이 어렵고 계산 비용도 크다는 한계점이 있었다. 

**📍 Related Work 2: 잠재변수 중점 생성모델 연구**

Autoencoder, Helmholtz Machine처럼 잠재변수를 이용하는 연구도 있었다. 이러한 방법들은 입력 데이터를 압축된 latent representation으로 바꾸고 다시 복원하는 구조를 사용하였다. 하지만 이 방식은 주로 입력 복원에 초점이 맞추어져 있어서, 실제 데이터 분포를 자연스럽게 생성하는 데에는 한계가 있을 수 있었다. 또한 잠재변수에 대한 추론 과정이 필요하다는 점에서도 복잡성이 존재했다.

**🔍 Methods**

✅ **사용된 연구 방법**: 이 논문에서는 GAN이라는 새로운 프레임워크를 제안한다. 기존 생성모델처럼 데이터 분포를 직접 계산하는 대신, 두 개의 신경망이 서로 경쟁하도록 학습시키는 방식을 사용한다. 하나는 가짜 데이터를 생성하는 Generator이고, 다른 하나는 입력 데이터가 진짜인지 가짜인지를 구분하는 Discriminator이다. Generator는 Discriminator를 속이도록 학습되고, Discriminator는 이를 더 잘 판별하도록 학습된다. 이를 통해 Generator는 점점 실제 데이터와 비슷한 샘플을 생성할 수 있게 된다.

**✅ 실험 설계:** 논문에서는 제안한 GAN이 실제로 생성모델로 동작하는지 확인하기 위해, 생성자와 판별자를 모두 신경망으로 구성한 뒤 여러 데이터셋에서 실험을 수행하였다. 실험의 목적은 생성된 샘플이 실제 데이터 분포를 얼마나 잘 따라가는지 확인하는 것이었다. 또한 단순히 샘플을 시각적으로 확인하는 것뿐만 아니라, 학습된 표현이 다른 과제에도 도움이 되는지를 함께 살펴보았다!

**📍 모델 비교**: 크게 다른 모델들과의 비교는 없었다. 대신 Background에 작성했던 모델들과의 비교는 있어 해당 부분 내용을 작성했다. 

- 기존 확률 기반 생성모델 : 복잡한 확률 계산과 샘플링 과정이 필요하다.
- Autoencoder 계열 모델 : 입력 복원에는 강점이 있지만, 생성 품질 측면에서 한계가 있을 수 있다.
- GAN : 별도의 복잡한 추론 없이, Generator와 Discriminator의 경쟁 구조를 통해 샘플을 생성한다 !

**🔍 Experiments**

✅ **데이터셋**: 논문에서는 활용한 데이터셋은 다음과 같다.

- MNIST : 손글씨 숫자 이미지 데이터셋
- Toronto Face Database (TFD) : 얼굴 이미지 데이터셋
- synthetic data : 모델의 기본 동작을 확인하기 위한 인공 데이터

GAN이 단순 숫자 데이터뿐만 아니라 복잡한 이미지 데이터에도 성능이 좋음을 보이고자 활용했다 볼 수 있다. 

**✅ Models:** 실험에서는 Generator와 Discriminator를 모두 기본적인 다층 퍼셉트론(MLP) 구조로 구성하였다.

**✅ Evaluation Metrics:** 이 논문은 최근 연구처럼 복잡한 정량 지표를 사용하지 않았다. 대신 생성된 이미지가 실제처럼 보이는지를 확인하는 정성적 평가를 중심으로 결과를 분석하였다.

**✅ Implementation Details:** Generator는 noise vector를 입력받아 가짜 데이터를 생성하고, Discriminator는 진짜와 가짜를 구분하도록 학습된다. 두 모델은 번갈아가며 업데이트하는 방식으로 학습되었다.

---

**📖 Conclusion**

✅ **Limitation:** GAN은 새로운 생성모델 학습 방법을 제시했지만, 학습이 불안정할 수 있다는 한계가 있다. 또한 생성자가 일종의 과적합마냥 비슷한 샘플만 반복해서 만드는 문제가 발생할 수도 있다.

**✅ Contribution:** 다음과 같다.

- GAN이라는 새로운 생성형 모델 관련 프레임워크 제안
- Generator와 Discriminator의 경쟁 구조를 통해 학습하는 방식 제시
- 복잡한 확률 계산 없이도 데이터 생성이 가능함을 보임
- 이후 다양한 GAN 연구의 기반이 됨

**Q. 왜 GAN의 최적 상태에서는 판별자가 1/2를 출력하게 되는가?**

A. 생성자가 실제 데이터 분포를 완벽하게 따라가게 되면, 판별자는 입력된 샘플이 진짜인지 가짜인지 더 이상 구분할 수 없게 된다. 따라서 어떤 입력이 들어와도 진짜일 확률과 가짜일 확률을 동일하게 보고 1/2를 출력하게 되는 것이다.
