# 📚 4주차 TASK

# 1️⃣ TASK 1 — 모델 디벨롭 (Model Development)

베이스라인 모델을 기반으로 **모델 성능을 개선**해 봅시다.

다음과 같은 방법을 활용할 수 있습니다.

- 하이퍼파라미터 튜닝
- 앙상블 기법 적용
- 추가 전처리
- 추가 피처 엔지니어링

모델을 여러 방식으로 디벨롭한 뒤,  
**각 모델의 성능 지표를 비교 분석**해 주세요.
